專案由來
------
	實作 PHP Laravel的Blog後，為了加深對MVC框架的了解，
	所以又學習用ASP.NET MVC 5 製作相同的Blog	

實作功能
------
	發表文章、編輯文章、刪除文章、留言、
	精選文章、亂數文章

學習成果
------
	1. 開發環境建置
	2. C# MVC 5 專案結構
	3. C#基本語法
	4. View - Partial、Bundle、BeginForm
	5. Model - migration、seeding、ViewModel	
	6. Controller - linq、HttpPost、ValidateAntiForgeryToken
